# Card Matching Game
Implementation of a simple card-matching game for a developer technical task.

## Tech
- Engine: Unity 2021 LTS
- Language: C#
  
## How to open and play
1. Clone the repository and open the project in Unity 2021 LTS
2. In the Unity Editor, open the main scene: "Assets/_Dev/Scene/Test.unity"
3. Press the Play button in the toolbar to start the game.
